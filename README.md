## 运行项目

### 安装 flutter

环境搭建流程参照：[安装 Flutter](https://flutter.dev/docs/get-started/install)

### 下载项目

```
git clone https://github.com/liyuandie/flutter_shop.git
```

### 运行

```
cd flutter_shop
flutter run
```
